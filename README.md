# Notes
Notes can help you to create notes.

The technologies, that are used:
- Servlets
- JSP, JSTL
- HTML, CSS, XML
- Maven
- Tomcat
- JUnit
- Mockito
- Jacoco

## Running
Using Tomcat to open the link: ~/index.html

Check tests through the Jacoco:
```
mvn clean package
```
And follow to: ~/target/site/jacoco/index.html
